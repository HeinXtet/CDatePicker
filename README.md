# CDatePicker
Android Date Edittext

Add Project Level Gradle

```
  allprojects {
	repositories {
		maven { url 'https://jitpack.io' }
		}
	   }
	
Add App Level Gradle

dependencies {
	   implementation 'com.github.HeinXtet:CDatePicker:1.0'
	}
```
